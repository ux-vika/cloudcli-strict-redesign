# Контекст: редизайн главного экрана CloudCLI (strict light/dark)

Этот документ — передача контекста из предыдущей сессии. Читай его первым.

## Что делаем
Редизайн **главного экрана CloudCLI** (сайдбар + чат/сессия) в визуальном
направлении **«strict»**, в двух режимах: **Strict Light** и **Strict Dark**.
Контент и раскладка элементов сохраняются 1:1 с оригиналом (v1.36.1) — меняется
только оформление: типографика, иконки, контролы, цвет.

## Ключевые решения (уже согласованы с пользователем)
- **С нуля, свежий клон оригинала.** Эта папка — чистый клон `siteboon/claudecodeui`
  (origin → оригинал, последний коммит апстрима). Никаких прошлых правок редизайна здесь нет.
- **НЕ трогаем** два других проекта пользователя:
  - `~/portfolio-deploy` — её портфолио, вообще не касаемся.
  - `~/Documents/redesign-claude-code` — её прежний форк с незавершённым редизайном, не касаемся.
- Реализация ведётся **в этой папке**, в отдельной сессии Claude, привязанной к ней
  (чтобы корректно работали git, `npm run dev` и превью в браузере).
- Подход к реализации по хендоффу: **воссоздать дизайн в существующей кодовой базе**
  (React + Vite + Tailwind), используя её компоненты, токены и паттерны. НЕ копировать
  прототипные SVG/HTML напрямую; иконки маппить на существующий набор (lucide-react),
  палитры завести в механизм тем приложения как пару light/dark.

## Источник истины (макеты и спека)
Лежат прямо здесь:

## Куда смотреть в коде (найдено при осмотре кодовой базы)
Компоненты главного экрана:
- Сайдбар: `src/components/sidebar/` (header, content, mode-nav, project-sessions, session-item).
- Чат/сессия: `src/components/chat/` (в т.ч. `view/subcomponents/ChatComposer.tsx`,
  `MessageComponent.tsx`), `src/components/main-content/`.
- Настройки: `src/components/settings/`.
- Темы/токены: `src/index.css`, `tailwind.config.js`.

## Краткая выжимка дизайн-токенов (детали — в HANDOFF-spec.md)
- Шрифты: UI — Instrument Sans (400/500/600/700); mono — JetBrains Mono (400/500/600).
- Радиусы: карточки/инпуты 10px; строки/кнопки 6px; бабблы 14/14/14/4 (ассистент) и 14/14/4/14 (юзер); пилюли 20px.
- Focus ring: 2px accent, offset 1px.
- **Strict Light**: accent `#1D4ED8`; canvas `#FFFFFF`; sidebar `#F7F8FA`; borders `#E2E5EA`;
  text primary `#111827`; user bubble bg `#EAF0FB`.
- **Strict Dark**: accent `#5B8DEF`; canvas `#0F1115`; sidebar `#12141A`; surfaces `#14161C`;
  borders `#262A33`; text primary `#F0F2F5`; user bubble bg `#243149`.

## Рекомендованные первые шаги в новой сессии
1. `npm install`, затем `npm run dev` — поднять приложение локально и увидеть текущий вид.
2. Изучить механизм тем (`src/index.css`, `tailwind.config.js`, контекст темы, если есть) —
   понять, как заведены light/dark и токены.
3. Через навык **brainstorming** финализировать дизайн-спеку → написать план (writing-plans) →
   реализовывать по шагам, сверяясь с макетами (скриншот → сравнение → правка).
4. Язык общения и коммитов — русский (правило пользователя).
