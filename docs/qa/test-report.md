## ❌ Найденные баги

### BUG-01 — «белый фон body в тёмной теме» → расследован, страховочный фикс внесён
- Глубокая проверка показала: правило `body { background-color: hsl(var(--background)) }` присутствует
  в собранном CSS, визуально тёмная тема всегда рендерилась корректно (подтверждено скриншотами).
  Аномальные замеры `getComputedStyle` — артефакт изолированного контекста тест-панели.
- На всякий случай в `src/index.css` добавлен явный фон для body (не только через `@apply`) —
  страхует от «белой вспышки» при загрузке. Плюс из `index.html` удалена загрузка неиспользуемых
  Google Fonts (Encode Sans, Merriweather) — редизайн полностью на self-hosted шрифтах.

### ~~BUG-02~~ — снят. Ложное срабатывание тест-стенда
Повторная проверка показала: модал Settings закрывался корректно (и по Esc, и по крестику) — «незакрывающийся модал» оказался закэшированным кадром скриншотера панели, в живом DOM модала не было. Багов не подтверждено.
