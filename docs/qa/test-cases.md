# Тест-кейсы CloudCLI (strict-редизайн)

Версия приложения: 1.36.1 + strict-редизайн.
Окружение: desktop-браузер (Chrome), dev-сборка `http://localhost:5180`, обе темы (Light/Dark).
Легенда приоритетов: **P0** — критично (ядро сценария), **P1** — важно, **P2** — желательно.

Для каждого кейса, если не сказано иное, предусловие: пользователь авторизован, открыт главный экран.

---
