# Карта проекта — куда идти, чтобы что-то поменять

Путеводитель по коду CloudCLI (strict-редизайн). Цель — **не открывать весь проект**, а сразу идти в нужный блок.

Код организован **по фичам**: каждая крупная часть интерфейса живёт в своей папке-модуле внутри [`src/components/`](../src/components/). Внутри модуля всё разложено одинаково — поэтому, зная одну папку, ты знаешь их все.

## Анатомия одного блока

На примере [`src/components/sidebar/`](../src/components/sidebar/):

```
sidebar/
├── view/                  ← ЧТО видно: React-компоненты
│   ├── Sidebar.tsx        ← «входная дверь» блока (главный компонент)
│   └── subcomponents/     ← части экрана (шапка, список, строка проекта…)
├── hooks/                 ← ЛОГИКА: состояние и поведение (useSidebarController)
├── utils/                 ← вспомогательные функции блока
├── types/                 ← типы TypeScript этого блока
└── index.ts              ← (не у всех) короткая точка входа для импорта
```

**Правило навигации:**
- меняешь **внешний вид / вёрстку** → `view/` (и `view/subcomponents/`)
- меняешь **поведение / данные / состояние** → `hooks/`
- меняешь **формат данных, форматирование** → `utils/`

## Общие папки ([`src/`](../src/)) — не относятся к одному блоку

| Папка | Что внутри |
|-------|-----------|
| [`shared/view/ui/`](../src/shared/view/ui/) | **UI-кит**: переиспользуемые примитивы — `Button`, `Dialog`, `Input`, `Tooltip`, `ScrollArea`, `PromptInput`, `ActionMenu`, `Card`, `Badge`… Меняешь тут — меняется везде |
| [`stores/`](../src/stores/) | Глобальное состояние (`useSessionStore`) |
| [`contexts/`](../src/contexts/) | React-контексты: `AuthContext`, `ThemeContext`, `WebSocketContext`, `PermissionContext`, `PluginsContext`, `TaskMasterContext`… |
| [`hooks/`](../src/hooks/) | Глобальные хуки: `useProjectsState`, `useDeviceSettings`, `useSessionProtection`, `useVersionCheck`, `useUiPreferences`… |
| [`lib/`](../src/lib/) | Низкоуровневое: `utils.js` (функция `cn()` для классов), голосовой ввод |
| [`utils/`](../src/utils/) | Прикладные утилиты: `api`, `clipboard`, `dateUtils`, `notificationSound` |
| [`constants/`](../src/constants/) | Константы: `branding`, `config` |
| [`types/`](../src/types/) | Общие типы: `app.ts`, `sharedTypes.ts` |
| [`i18n/locales/`](../src/i18n/locales/) | **Переводы** по неймспейсам: `en/*.json`, `ru/*.json` |
| [`index.css`](../src/index.css) | Глобальные стили, **дизайн-токены** (CSS-переменные тем), transition-классы |
| [`App.tsx`](../src/App.tsx) / [`main.jsx`](../src/main.jsx) | Точка входа приложения |
| [`tailwind.config.js`](../tailwind.config.js) | Настройка Tailwind: цвета-токены, шрифты, **keyframes анимаций** |

## Куда идти для типичной задачи

| Хочу поменять… | Иду сюда |
|----------------|----------|
| Внешний вид **сайдбара** (проекты, чаты, навигация) | [`sidebar/view/subcomponents/`](../src/components/sidebar/view/subcomponents/) |
| Поведение сайдбара (открытие, поиск, состояние) | [`sidebar/hooks/useSidebarController.ts`](../src/components/sidebar/hooks/useSidebarController.ts) |
| **Композер** чата (поле ввода, кнопки, меню режима) | [`chat/view/subcomponents/ChatComposer.tsx`](../src/components/chat/view/subcomponents/ChatComposer.tsx) |
| Логику ввода композера (черновики, команды) | [`chat/hooks/useChatComposerState.ts`](../src/components/chat/hooks/useChatComposerState.ts) |
| Как рисуется **сообщение** в чате | [`chat/view/subcomponents/MessageComponent.tsx`](../src/components/chat/view/subcomponents/MessageComponent.tsx) |
| **Цвета, темы, дизайн-токены** | [`src/index.css`](../src/index.css) (CSS-переменные) + [`tailwind.config.js`](../tailwind.config.js) |
| **Анимации / переходы** | [`tailwind.config.js`](../tailwind.config.js) (keyframes) + [`src/index.css`](../src/index.css) (transition-классы) |
| Общую **кнопку / диалог / инпут** | [`shared/view/ui/`](../src/shared/view/ui/) |
| **Тексты и переводы** | [`i18n/locales/en/`](../src/i18n/locales/en/) и [`i18n/locales/ru/`](../src/i18n/locales/ru/) |
| **Настройки** (экран настроек) | [`settings/`](../src/components/settings/) |
| **Git-панель** | [`git-panel/`](../src/components/git-panel/) |
| **Терминал** | [`shell/`](../src/components/shell/) |
| **Дерево файлов** | [`file-tree/`](../src/components/file-tree/) |

## Самые большие файлы (навигация внутри крупного)

Если файл большой — не листай целиком, ищи по названию функции/компонента:

| Файл | Строк | Про что |
|------|-------|---------|

| [`hooks/useProjectsState.ts`](../src/hooks/useProjectsState.ts) | ~1090 | Состояние проектов и сессий |
| [`sidebar/hooks/useSidebarController.ts`](../src/components/sidebar/hooks/useSidebarController.ts) | ~1000 | Поведение сайдбара |
| [`chat/hooks/useChatSessionState.ts`](../src/components/chat/hooks/useChatSessionState.ts) | ~860 | Состояние сессии чата |
| [`sidebar/view/subcomponents/SidebarContent.tsx`](../src/components/sidebar/view/subcomponents/SidebarContent.tsx) | ~765 | Списки проектов/чатов/поиска в сайдбаре |
| [`chat/view/subcomponents/ChatComposer.tsx`](../src/components/chat/view/subcomponents/ChatComposer.tsx) | ~680 | Разметка композера |
